**You can filter:**
- Min price
- Max price
- Max supply
- Network *(work in progress, for now only Binance Smart Chain)*
- By date added *(to coinmarket cap)*

## Development

Fist set you `.env` file, this required a [CoinMarketCap API KEY](https://coinmarketcap.com/api/)

```bash
$ cp .env.example .env
```

```bash
$ npm install
```

```bash
$ npm dev
```

## Production

```bash
$ npm install
```

```bash
$ npm start
```
