Is a 4 hours project to check the newest currencies with zero effort.


You can filter:

- By date added *(to coinmarket cap)*

## Development

Fist set you `.env` file, this required a [CoinMarketCap API KEY](https://coinmarketcap.com/api/)

```bash
$ cp .env.example .env
```

```bash
$ npm install
```

```bash
$ npm dev
```

## Production

```bash
$ npm install
```

```bash
$ npm start
```
